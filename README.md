# Automation case studies

Write-ups of internal systems I've built at FunNow Group (the company behind Eatigo). The code is internal; the patterns aren't. Each study covers what the problem looked like, how the system works, and what I'd do differently.

I come from the business side (lifecycle and CRM), and I build these systems myself with Claude Code, n8n, and Supabase. That shapes the writing: less about elegant code, more about what made operators actually use the thing.

## Studies

1. [The lead-gen agent: scrape, dedupe, outreach, CRM](01-lead-gen-agent.md): an always-on B2B pipeline that BD runs in three steps
2. [Campaign auto-setup: from hours to minutes](02-campaign-auto-setup.md): one form that generates push and EDM campaigns across 7 markets, and gives the team back about 84 hours a month

More coming. The backlog: the e-signature integration that closes deals inside our own CRM, and how a non-engineer ships production tools with Claude Code as the build partner.
